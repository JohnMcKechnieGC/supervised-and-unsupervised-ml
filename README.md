# Supervised and Unsupervised Machine Learning
This project contains two supervised machine learning demos:
- classifying-penguins.ipynb - an in-depth tutorial using the DecisionTreeClassifier from scikit-learn
- classifying-irises.ipynb - exploring DecisionTreeClassifier and RandomForestClassifier

It also includes:
-  k-means-clustering-demo.ipynb - a K-means clustering demo that first demonstrates the algorithm step-by-step, and then uses KMeans from scikit-learn
- hierarchical-clustering.ipynb - demonstrating the linkage function from SciPy

The evaluation.ipynb notebook demonstrates the leukaemia classifier example from Data Science from Scratch by Joel Grus (2015), O'Reilly Media, Inc. The classifier achieves high accuracy despite being utterly useless. See the classifying penguins tutorial for more detailed information on evaluating classifiers.
